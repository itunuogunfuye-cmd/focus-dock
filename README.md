# Focus Dock

Focus Dock is a smart desk accessory designed to reduce phone distractions during study or work sessions.

The system combines hardware and software to encourage focused behaviour through visual feedback, audio prompts, Bluetooth communication, and session tracking.

## Features

- Phone detection using FSR sensor
- Magnetic wireless charging
- LED progress feedback
- Audio and voice notifications
- Bluetooth communication (HC-05)
- Flutter mobile application
- Session statistics and history tracking

## Hardware

- Arduino Uno R3
- HC-05 Bluetooth Module
- DFPlayer Mini
- WS2812B LED Strip
- FSR Sensor
- Potentiometer
- Push Button
- Buzzer
- Magnetic Wireless Charger
